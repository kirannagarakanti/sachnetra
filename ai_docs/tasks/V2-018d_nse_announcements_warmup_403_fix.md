- ~**2026-05-30** Akamai tightened: the API now requires session cookies **`bm_sv` + `_abck`** that are *only* set when a **content page** is loaded (a 2nd hop). The single-hop warm-up only gets homepage cookies (`ak_bmsc`, `bm_sz`, `bm_mi`) â†’ insufficient â†’ 403.
- **NOT a hard datacenter-IP block.** Akamai uses a *behavioral threat score*: a datacenter IP polling **hourly** (Announcements) tripped the threshold â†’ 403 even on the homepage; the **once-daily** Deals collector stayed under it and still works from the same Railway IP. â‡’ **No proxy needed** if cadence is lowered.
- Two extra hardening findings: NSE now expects modern **client-hint / fetch-metadata headers**; and our **Chrome/120 UA is ~2.5 y stale** (current stable = Chrome 149), which raises the bot score on its own.

`fetchNSEAnnouncements()` row mapping is correct; only the **handshake (warm-up + request headers)** is broken. `warmUpNSE()`'s signature (`() => Promise<string cookie>`) stays identical, so `seed-india-announcements.mjs` and `backfill-india-announcements.mjs` (the only importers) need no changes.

```javascript
export async function warmUpNSE() {
  const resp = await fetch(NSE_ORIGIN, {
    headers: { 'User-Agent': CHROME_UA, Accept: 'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8' },
    signal: AbortSignal.timeout(20_000),
  });
  if (!resp.ok) throw new Error(`NSE warm-up HTTP ${resp.status}`);
  const setCookies = typeof resp.headers.getSetCookie === 'function'
    ? resp.headers.getSetCookie() : [resp.headers.get('set-cookie')].filter(Boolean);
  const cookie = setCookies.map((c) => String(c).split(';')[0].trim()).filter(Boolean).join('; ');
  if (!cookie) throw new Error('NSE warm-up returned no Set-Cookie (bot wall or markup change)');
  return cookie;
}
```

**After** (two hops, full Chrome-149 header set, merged cookies â€” incl. `bm_sv`/`_abck`):
```javascript
export async function warmUpNSE() {
  const resp1 = await fetch(NSE_ORIGIN, {
    headers: { 'User-Agent': CHROME_UA, Accept: NAV_ACCEPT, ...BASE_HINTS,
      'Upgrade-Insecure-Requests': '1', 'Sec-Fetch-Site': 'none', 'Sec-Fetch-Mode': 'navigate',
      'Sec-Fetch-User': '?1', 'Sec-Fetch-Dest': 'document' },
    signal: AbortSignal.timeout(20_000),
  });
  if (!resp1.ok) throw new Error(`NSE warm-up hop 1 HTTP ${resp1.status}`);
  const cookies1 = extractCookies(resp1.headers);

  const resp2 = await fetch(NSE_REFERER, {  // content page â†’ sets bm_sv / _abck / nsit
    headers: { 'User-Agent': CHROME_UA, Accept: NAV_ACCEPT, ...BASE_HINTS,
      'Upgrade-Insecure-Requests': '1', Cookie: cookies1, Referer: NSE_ORIGIN,
      'Sec-Fetch-Site': 'same-origin', 'Sec-Fetch-Mode': 'navigate',
      'Sec-Fetch-User': '?1', 'Sec-Fetch-Dest': 'document' },
    signal: AbortSignal.timeout(20_000),
  });
  if (!resp2.ok) throw new Error(`NSE warm-up hop 2 HTTP ${resp2.status}`);
  const cookies2 = extractCookies(resp2.headers);

  const merged = mergeCookies(cookies1, cookies2);
  if (!merged) throw new Error('NSE warm-up returned no Set-Cookie (bot wall or markup change)');
  return merged;
}
```
