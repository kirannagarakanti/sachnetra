## Issues

Two issues found during browser verification of Task 013:

### Issue 1: State bar visible on Timeline tab

The "All India / Change state" bar (`snStateBar`) and the state grid (`snStateGrid`) are rendered as **siblings** to `snFeed` (lines 343‚Äì399 of `panel-layout.ts`), not inside it. When `activateTab('timeline')` hides `snFeed`, the state bar remains visible because it's a separate DOM element.

**Fix**: In the `activateTab()` function inside `setupMobileIndiaLayout()`, hide `snStateBar` and `snStateGrid` on all tabs except `home`. The state bar is only relevant to the Home tab (which shows state-filtered story cards). Timeline is all-India by design.

### Issue 2: Font sizes too small on Timeline

The mockup uses readable sizes but the current implementation uses very small fonts:
- `.sn-tl-title` is `12px` ‚Äî bump to `13px`
- `.sn-tl-source` is `10px` ‚Äî bump to `11px`
- `.sn-tl-ago` is `10px` ‚Äî keep (metadata, smaller is fine)
- `.sn-tl-chip` is `11px` ‚Äî bump to `12px`, increase padding
- `.sn-tl-divider-text` is `10px` ‚Äî bump to `11px`
