Audio-enabled VVPAT system sought for visually impaired voters
WHAT HAPPENED
The Election Commission of India is considering introducing an audio-enabled VVPAT system to help visually impaired voters cast their ballots. This would allow voters to hear candidate names through audio output, making voting more accessible.
