## The two actionable findings (both fixable with data we already have)
1. **Taxonomy gap — no block-deal/ownership category.** BOTH C-grades (p1d-21 Lenskart, p1d-28 GNG) are
   secondary **block deals / promoter stake-sales** mislabeled `mna`. The model had no better option in the
   closed list, and p1d-28's own rationale even says "to meet public shareholding norms" — contradicting its
   own `mna` label. **Fix: add an `ownership`/`block_deal` event_type.** SachNetra already collects bulk/block
   deals (V2-030), so this category is first-class. This single fix would have made the run 30/30 pass-grade.
2. **Unlisted-entity handling.** p1d-01 vs p1d-29 (two Tata Sons board articles) split — one tagged the
   unlisted holding company as `primary_entity`, one correctly returned `none`. The model can't reliably know
   Tata Sons is unlisted. **Fix: validate `primary_entity` against the NSE equity master** (`shared/nse-equity-master.json`,
   already live) — if the extracted name isn't in the listed universe, demote to `none`/watchlist. Deterministic,
   zero model cost, and it also catches private-arm attributions like p1d-30.
